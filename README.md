## Run Locally

Clone the project

```bash
  git clone https://github.com/ozgeahras/my-emoji-app.git
```

Go to the project directory

```bash
  cd ..my-emoji-app
```

Install dependencies

```bash
npm install
```

Start the app

```bash
npm run start
```
